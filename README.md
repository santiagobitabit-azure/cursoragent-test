# cursoragent-test

Cuenta regresiva hasta el primer partido de la Selección Argentina en el Mundial 2026.

- **Partido:** Argentina vs Argelia
- **Fecha:** martes 16 de junio de 2026, 22:00 (hora Argentina)
- **Sede:** Arrowhead Stadium, Kansas City

## Ver en local

Abrir `index.html` en el navegador, o servir la carpeta:

```bash
python3 -m http.server 8080
```

Luego visitar http://localhost:8080
